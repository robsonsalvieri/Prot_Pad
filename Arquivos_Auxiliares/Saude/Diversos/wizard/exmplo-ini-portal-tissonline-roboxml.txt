Para funcionamento correto do PORTAL a pasta WEB (PP e WS) devem estar 
atualizadas.

Esta pasta contem arquivos de IMAGENS, JAVASCRIPT e CSS.
Arquivos JS e CSS da pasta PP devem estar atualizados.
Arquivos JS e CSS da pasta IMAGENS-PLS devem estar atualizados.

Estes arquivos estão disponíveis em 
https://suporte.totvs.com/download, linha Protheus, filtro 
(Arquivos de Portais)

*** Exemplo de configuração

---------------------------------------------------------------------------
ONSTART
---------------------------------------------------------------------------
[ONSTART]
JOBS=JOB_WS,JOB_PP,XML_ROBOT
RefreshRate=160

[XML_ROBOT] 
ENVIRONMENT=ENV-LOCAL
main=xmlroute 
---------------------------------------------------------------------------
ENVIRONMENT
---------------------------------------------------------------------------
[ENV-LOCAL]	

Configuração da Environment 

---------------------------------------------------------------------------
HOSTS PP e TISS ONLINE

Host de acesso interno do portal para utilizar externamente basta copiar
esta sessão
substituir LOCALHOST pelo domínio ou IP
---------------------------------------------------------------------------

[LOCALHOST]
ENABLE=1

Informar pasta com conteúdo WEB PP
PATH=C:\MP11\DATA\WEB\PP

INSTANCENAME=PLS
RESPONSEJOB=JOB_PP
RpcEnable=0
RpcEnv=ENV-LOCAL
RpcTimeout=0
Cache-Control=no-store
Pragma=no-cache 

Informar pasta para Upload de arquivos
UPLOADPATH=\Tiss\UpLoad

Não exibe o combobox para seleção de portais (Portal Protheus entre outros)
TPACESSO=1

---------------------------------------------------------------------------
Host para acesso TISS-ONLine
---------------------------------------------------------------------------
[LOCALHOST/TISSON]
ENABLE=1
Informar a pasta contendo SCHEMAS
PATH=C:\MP11\Data\tiss\schemas

INSTANCENAME=PLS
RESPONSEJOB=JOB_TISS
Cache-Control=no-store
Pragma=no-cache 

---------------------------------------------------------------------------
HOSTS WS
---------------------------------------------------------------------------
[LOCALHOST/WS]
ENABLE=1
Informar pasta com conteúdo WEB WS
PATH=C:\MP11\DATA\WEB\WS

ENVIRONMENT=ENV-LOCAL
INSTANCENAME=WS
RESPONSEJOB=JOB_WS
DEFAULTPAGE=wsindex.apw
---------------------------------------------------------------------------
JOBS - Atende a requisição PP,WS e TISS ONLine
Atende WS
---------------------------------------------------------------------------

[JOB_WS]
TYPE=WEBEX
ENVIRONMENT=ENV-LOCAL

Instancias de processamento para WS. Verificar configuração no 
TDN http://tdn.totvs.com/display/tec/Instances

INSTANCES=1,1

SIGAWEB=WS
INSTANCENAME=WS
ONSTART=__WSSTART
ONCONNECT=__WSCONNECT

Informe a empresa e a filial
PREPAREIN=99,01

---------------------------------------------------------------------------
Atende Portal 
---------------------------------------------------------------------------
[JOB_PP]
TYPE=WEBEX
ENVIRONMENT=ENV-LOCAL

Instancias de processamento para WS. Verificar configuração 
no TDN http://tdn.totvs.com/display/tec/Instances

INSTANCES=1,1

SIGAWEB=PP
INSTANCENAME=PLS
ONSTART=STARTWEBEX
ONCONNECT=CONNECTWEBEX
ONEXIT=FINISHWEBEX
INACTIVETIMEOUT=1800
EXPIRATIONTIME=3600
EXPIRATIONDELTA=600
WEBSERVICELOCATION=HTTP://LOCALHOST/WS
DebugMsg=1

---------------------------------------------------------------------------
Atende TISS ONLine
---------------------------------------------------------------------------
[JOB_TISS]
TYPE=WEBEX
ENVIRONMENT=ENV-LOCAL

Instancias de processamento para WS. Verificar configuração 
no TDN http://tdn.totvs.com/display/tec/Instances

INSTANCES=1,1

SIGAWEB=MAK
ONSTART=STARTWEBEX
ONCONNECT=CONNECTWEBEX
ONEXIT=FINISHWEBEX

---------------------------------------------------------------------------
HTTP
---------------------------------------------------------------------------
[HTTP]
ENABLE=1
PORT=80

Especifica 20 minutos de Time-Out 
SESSIONTIMEOUT=1200
